Tool Name Android Studio 3.0.1
Platform Android
App Development Type Native, which is good because apps can accessed offline, but bad because apps require each user to download.
Website https://developer.android.com/studio/index.html
Developers Google and JetBrains
Release Date December 2014
License https://developer.android.com/studio/terms.html
Price Free
Programming Language(s) Android Studio primarily uses Java.
Features Instant Run, intelligent code editor, emulator, can integrate with version control tools, optimized for all Android devices, etc.
Debugger The debugger allows you to select a device to debug on, set breakpoints and analyse variables at runtime.
Built-in Emulator The build-in emulator allows you to test on vaarious Android device configurations and allows simulation of GPS, network latency, multi-touch and motion sensors.
Host Machine Requirements Windows 7 or higher, Mac OS X 10.10 or higher, for Linux - GNOME OR KDE. 3GB RAM, 2GB available space and 1280x800 screen resolution are the minimum requirements.
IDE Built-in, user-friendly IDE with templates and other features.
Required Software Android Studio IDE, SDK, JDK and emulator.
Advantages Most widely used and supported Android IDE, built-in emulator, can be used with version-control programs, all-in-one installation, open source and free.
Disadvantages Slow build times, some users claim it has a lot of bugs and it requires the developer to have a good understand of how to code.
Language Availability Android Studio supports many different languages and locales, including English, German, Danish, Japanese, Chinese, Hindi, French, Russian, and much more.
Uploading And Selling Apps Apps can be published through the app marketplace, primarily Google Play, which allows in-app billing and application licensing. Google Play has a one-time fee of $25 and they keep 30% of all profit earned from published apps.
Third-Party Emulator Genymotion is an Android emulator which can be installed as an Android Studio plugin. This emulator runs a lot faster than the built-in emulator, allows for console commands, has a scrolling feature and has a very user-friendly, accessable UI.
Market Share Android is the most popular mobile platform for app downloads. It is unknown how many were developed on Android Studio.
Number of Apps Available There are over 3 million apps on the Google Play store as of June 2017, it is unknown how many of those apps were created using Android Studio.
Number of Downloads/Users N/A
Performance Profilers Android Studio has built-in profiling tools and provide various usage and activity statistics from your apps in realtime.
Organization and Team Usage Android Studio allows direct integration with version control tools like GitHub and Subversion. These tools make app development much easier to manage in organizations and teams.
